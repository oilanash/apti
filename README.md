# Bissultanov's Poem about Khaybah

## About the Project

This project is dedicated to preserving one of the most important works of Chechen literature — Bissultanov's poem about Khaybah. A work that is impossible to translate because it speaks to the hidden corners of the mind, directly connecting us to the history of the people, bypassing reason and logic. A translation would destroy this connection — only words would remain, while the impact would vanish.

The work is presented in its original Chechen language, without translations. This is a tribute to the uniqueness of the language and the poem itself.

## Project Purpose

This project was created to preserve the original text of the poem in an accessible and lasting form, so that future generations who seek it will be able to find it unchanged. I decided to host it on GitHub and also take advantage of the opportunity to preserve it in the Arctic Code Vault.

The Arctic Code Vault, located on Svalbard, is a place where data recorded on silver halide film will be stored for a thousand years, surviving any server and any government that ever decides what is worthy of remembrance. This is a unique opportunity to preserve not only technology but also cultural heritage.

## Project History

It all started when a friend asked me to share a link to the poem. I opened the browser and discovered that it was not available. The internet, which stores every meme and every scandal, did not preserve one of the most important texts written in the Chechen language. I decided to correct that.

The project was not planned, but it became an important part of my journey. The link didn’t exist, but now the text will be preserved in the Arctic ice for a thousand years, in a place where its memory will not fade.

## How It Works

The poem is stored in the repository in its original Chechen text. The data has also been sent to the Arctic Code Vault, which guarantees its preservation for a thousand years in the conditions of perpetual frost, inside an abandoned coal mine, behind the steel doors of the Arctic repository.
